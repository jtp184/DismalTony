Generates an API request from the provided data to this mixin. Parses and returns the JSON body of a request, intelligently merging given arguments in input_args and the api_defaults

[show source]

def api_request(input_args)
  addr = URI(api_url)
  parms = input_args.clone
  parms = api_defaults.merge(parms)
  addr.query = URI.encode_www_form(parms)
  JSON.parse(Net::HTTP.get(addr))
end
