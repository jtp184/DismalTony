add_criteria ()

Yields the criteria array, and uses the block to add its results to match_criteria afterwards, allowing you to add new MatchLogic objects using the DSL.

def add_criteria
  crit = []
  yield crit
  @match_criteria ||= []
  @match_criteria += crit
end
