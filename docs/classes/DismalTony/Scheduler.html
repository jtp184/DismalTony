execute ()

Main resolution method. Iterates through the events, and handles them depending on event state. Deletes successfully finished or failed events, calling @failure_method if it exists

[show source] 
# File lib/dismaltony/scheduler.rb, line 26
def execute
  events.each do |event|
    case event.state
    when :waiting
      event.run(@vi) if event.ready?
    when :success
      events.delete(event)
    when :failure
      failure_method.call if @failure_method
      events.delete(event)
    end
  end
end
